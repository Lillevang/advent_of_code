# Advent of Code solutions

This repository holds all my solutions for advent of code. This is work in progress and a large amount of cleanup is overdue.

In some of the days there will be multiple solutions. I try to be disciplined about naming my own solutions `solution` - sometimes with a number after if I iterate over it a few times.
Solutions named `sol`are generally not my own, but solutions I've received or found that I wanted to consider for refactoring in the overdue cleanup activity.

This repo is a reupload of my original. Per Eric Wastl's (creator of Advent of Code) wishes I pruned the repo for all input data and deleted the original repo to remove the git history with it.
